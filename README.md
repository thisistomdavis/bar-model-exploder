# Bar Model Exploder! 💥

An interactive educational game designed to help children (particularly Year 3 and above) visualize and master addition using the **Bar Model** method.

## 🚀 Overview

The Bar Model Exploder takes standard addition problems and "explodes" them into their constituent parts—Hundreds, Tens, and Ones. This visual approach helps students understand the mechanics of partitioning and exchanging (carrying) in a fun, engaging environment.

## ✨ Features

- **Dual Modes:**
  - **Simple Mode:** Basic partitioning without exchanging.
  - **Exchanging Mode:** Advanced problems requiring "regrouping" (e.g., exchanging 10 ones for 1 ten).
- **Interactive Explosions:** Watch the numbers break apart into place value chunks!
- **Space-Themed Rewards:** Successful answers launch a rocket to the moon! 🌙
- **Detailed Workings:** Generates a step-by-step math report.
- **Printable Reports:** Easy "Print Workings" button for school portfolios or homework.
- **Vanilla Tech:** Built entirely with HTML, CSS, and JavaScript—no heavy frameworks required.

## 🛠️ How to Use

1. **Choose your mode:** Use the toggle at the top to switch between "Simple" and "Exchanging".
2. **Solve the problem:** Look at the two bottom bars and type the total sum into the top bar.
3. **EXPLODE!:** Click the button to see the partitioning animation.
4. **Review:** Read the "My Math Workings" section to see exactly how the numbers were combined and where exchanges happened.
